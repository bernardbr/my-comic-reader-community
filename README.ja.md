## ようこそ、読者の皆様！

私は **Bernardo** です。漫画の世界を愛し、無料でオープンソースのソフトウェアの力を信じている開発者です。このスペースは、私たちが一緒に最高の読書体験を構築できるように作成されました。

ここは、ユーザー、ファン、貢献者が協力し、アイデアを共有し、システムを改善するための専用スペースです。メインツールは誰でも無料で利用できるため、皆様からのフィードバックが開発の原動力となります。

💖 **[GITHUB スポンサーになる](https://github.com/sponsors/bernardbr)**

## 🚀 参加方法

このリポジトリは主に次の2つの目的で使用されます：

### 1. 🐛 バグと問題の報告
不具合を見つけましたか？期待通りに動作しませんか？
* [Issues](https://github.com/bernardbr/my-comic-reader-community/issues) タブをチェックして、既に報告されていないか確認してください。
* 報告がない場合は、[新規 Issue を作成](https://github.com/bernardbr/my-comic-reader-community/issues/new) してください。
* 可能な限り詳細な情報（ログ、スクリーンショット、再現手順）を提供してください。

### 2. 💡 機能リクエストとディスカッション
新機能の素晴らしいアイデアがありますか？ロードマップについて話し合いたいですか？
* [Discussions](https://github.com/bernardbr/my-comic-reader-community/discussions) タブへ移動してください。
* 気に入ったアイデアに投票したり、新しいスレッドを作成してコミュニティからのフィードバックを集めましょう。
