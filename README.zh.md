## 🚀 如何参与

我们主要将此仓库用于两件事：

### 1. 🐛 报告缺陷与问题
发现了漏洞？某些功能运行不正常？
* 查看 [Issues](https://github.com/bernardbr/my-comic-reader-community/issues) 选项卡，确认是否已有报告。
* 如果没有，请 [提交新 Issue](https://github.com/bernardbr/my-comic-reader-community/issues/new)。
* 请提供尽可能详尽的信息（日志、截图和复现步骤）。

### 2. 💡 功能需求与讨论
对新功能有绝佳的创意？想聊聊开发路线图吗？
* 前往 [Discussions](https://github.com/bernardbr/my-comic-reader-community/discussions) 选项卡。
* 为您喜欢的想法投票，或发起新话题以收集社区反馈。
