## Welcome, Reader!

I'm **Bernardo**, a developer who loves the world of comics and believes in the power of free, ad-free software. This space was created so we can build the best reading experience together.

This is a dedicated space for users, enthusiasts, and contributors to collaborate, share ideas, and help improve the system. Since the main tool is free for everyone, your feedback is what drives our development.

## 🚀 How to Participate

We use this repository primarily for two things:

### 1. 🐛 Reporting Bugs & Issues
Found a glitch? Something not working as expected?
* Check the [Issues](https://github.com/bernardbr/my-comic-reader-community/issues) tab to see if it has already been reported.
* If not, [Open a New Issue](https://github.com/bernardbr/my-comic-reader-community/issues/new).
* Please provide as much detail as possible (logs, screenshots, and steps to reproduce).

### 2. 💡 Feature Requests & Discussions
Have a great idea for a new feature? Want to chat about the roadmap?
* Head over to the [Discussions](https://github.com/bernardbr/my-comic-reader-community/discussions) tab.
* Upvote ideas you like or start a new thread to gather feedback from the community.
