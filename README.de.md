# My Comic Reader Community 📚💬

Willkommen im offiziellen Community-Repository für **my-comic-reader**! 

Dies ist ein spezieller Ort für Nutzer, Enthusiasten und Mitwirkende, um zusammenzuarbeiten, Ideen auszutauschen und das System zu verbessern. Da das Haupt-Tool für alle kostenlos ist, treibt Ihr Feedback unsere Entwicklung voran.

---

## 🚀 Wie man teilnimmt

Wir nutzen dieses Repository hauptsächlich für zwei Dinge:

### 1. 🐛 Fehler & Probleme melden
Einen Fehler gefunden? Etwas funktioniert nicht wie erwartet?
* Prüfen Sie den Tab [Issues](https://github.com/bernardbr/my-comic-reader-community/issues), um zu sehen, ob es bereits gemeldet wurde.
* Wenn nicht, [Öffnen Sie ein neues Issue](https://github.com/bernardbr/my-comic-reader-community/issues/new).
* Bitte geben Sie so viele Details wie möglich an (Logs, Screenshots und Schritte zur Reproduktion).

### 2. 💡 Funktionsanfragen & Diskussionen
Haben Sie eine tolle Idee für eine neue Funktion? Möchten Sie über die Roadmap chatten?
* Gehen Sie zum Tab [Discussions](https://github.com/bernardbr/my-comic-reader-community/discussions).
* Stimmen Sie für Ideen ab, die Ihnen gefallen, oder starten Sie ein neues Thema, um Feedback von der Community zu erhalten.
